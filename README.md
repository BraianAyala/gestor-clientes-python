# gestor-clientes-python
Proyecto final realizado como parte de un curso de Python, orientado a integrar
conceptos fundamentales del lenguaje mediante el desarrollo de una aplicación
de gestión de clientes en consola.

## Funcionalidades
- Alta, baja y modificación de clientes
- Persistencia de datos
- Validaciones de información
- Menú interactivo por consola

## Tecnologías utilizadas
- Python 3

## Contexto
Proyecto desarrollado a partir de una consigna académica/propuesta de curso,
implementado como práctica integral de programación en Python.
